# 刷题工具

一个简单高效的网页刷题工具，支持快速答题、错题记录和复习功能。

## 功能特性

- ✅ **快速答题模式** - 支持单选题和多选题，即时反馈
- ✅ **错题记录** - 自动记录错误答案，便于后续复习
- ✅ **复习模式** - 专门针对错题的复习功能
- ✅ **统计面板** - 显示学习进度、准确率和错题数量
- ✅ **键盘快捷键** - 支持键盘操作，提高答题效率
- ✅ **数据导出/导入** - 备份和恢复学习数据

## 快速开始

### 1. 启动本地服务器

由于浏览器的安全限制（CORS），需要通过本地服务器运行此应用。

**方法一：使用 Python（推荐）**

```bash
# Python 3
cd /Users/niksun/Documents/study
python3 -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

**方法二：使用 VS Code Live Server**

1. 安装 VS Code 扩展：Live Server
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

**方法三：使用 Node.js**

```bash
# 安装 http-server
npm install -g http-server

# 启动服务器
cd /Users/niksun/Documents/study
http-server -p 8000
```

### 2. 打开浏览器

服务器启动后，在浏览器中访问：

```
http://localhost:8000
```

## 使用说明

### 首页（Dashboard）

- 查看所有科目的学习统计
- 点击"开始练习"进入练习模式
- 点击"复习错题"复习之前答错的题目
- 使用"导出数据"备份学习进度
- 使用"导入数据"恢复之前的学习数据

### 练习模式

**鼠标操作：**
- 点击选项选择答案
- 点击"提交"按钮提交答案
- 点击"下一题"/"上一题"按钮导航
- 点击右上角按钮显示/隐藏题目导航面板
- 在导航面板中点击题号直接跳转到该题

**键盘快捷键：**
- `A/B/C/D` - 选择对应选项
- `Enter` - 提交答案
- `N` - 下一题
- `P` - 上一题
- `V` - 显示/隐藏导航面板
- `Esc` - 退出练习模式

**题目导航面板：**
- 右侧面板显示所有题目的状态
- 灰色 (•) - 未答题
- 绿色 (✓) - 答对
- 红色 (✗) - 答错
- 蓝色边框 - 当前题目
- 点击任意题号可直接跳转
- 面板状态会自动保存，下次打开时保持

### 复习模式

- 只显示之前答错的题目
- 答对后自动从错题列表中移除
- 使用方式与练习模式相同
- 导航面板只显示错题

## 题目文件格式

应用读取以下 JSON 文件：

- `马克思主义基本原理.json`
- `中国近代史纲要.json`
- `习近平新时代.json`
- `中国近代史纲要_2.json`

文件应放在与 `index.html` 相同的目录下。

## 数据存储

- 使用浏览器的 `localStorage` 存储数据
- 数据包括：错题记录、学习统计
- 数据仅存储在本地浏览器中
- 清除浏览器数据会丢失学习记录（建议定期导出备份）

## 故障排除

### 题目无法加载

**问题：** 提示"加载题目失败"

**解决方案：**
1. 确认 JSON 文件在正确的位置
2. 确认通过本地服务器运行（不是直接打开 HTML 文件）
3. 检查浏览器控制台的错误信息

### 数据丢失

**问题：** 学习数据突然消失

**解决方案：**
1. 检查是否清除了浏览器缓存/数据
2. 使用"导入数据"功能恢复之前的备份
3. 建议定期使用"导出数据"功能备份

### 键盘快捷键不工作

**问题：** 按键没有反应

**解决方案：**
1. 确认当前在练习/复习模式页面
2. 点击页面确保焦点在页面上
3. 检查是否有其他应用占用了相同的快捷键
