# 刷题工具

一个功能完整的网页刷题工具，支持多种学习模式、错题管理、进度追踪和移动端适配。

## 功能特性

### 学习模式

- **练习模式** - 逐题答题，即时反馈，支持单选题、多选题和简答题
- **复习模式** - 专项复习错题，答对后自动从错题列表移除
- **背题模式** - 直接浏览题目答案和解析，无需交互，高效记忆知识点

### 题目类型

- **单选题**（type 0）- 单选按钮，支持键盘 A/B/C/D 选择
- **多选题**（type 1）- 复选框，支持多项选择
- **简答题**（type 3）- 点击或按 Enter/Space 显示答案，不计入错题统计

### 导航与进度

- **题目导航面板** - 右侧可折叠面板，显示所有题目状态，支持直接跳题
  - 灰色 (•) - 未答题
  - 绿色 (✓) - 答对
  - 红色 (✗) - 答错
  - 蓝色 (📖) - 简答题已查看
  - 蓝色边框 - 当前题目
- **进度面板（Dashboard）** - 显示各科目完成数、正确率、错题数及总体统计

### 数据持久化

- **答题状态持久化** - 答题结果跨会话保存，重新打开浏览器后恢复
- **答案选择恢复** - 返回已答题目时自动恢复之前的选择状态
- **数据导出/导入** - 备份和恢复全部学习数据（含答题记录和选择内容）
- **错题追踪** - 自动记录错误答案，支持跨科目管理

### 界面与体验

- **移动端适配** - 响应式布局，触控友好，最小 44px 触控目标
- **键盘快捷键** - 全键盘操作支持，提高答题效率
- **解析显示** - 答题后显示题目解析（如有）

## 快速开始

### 1. 启动本地服务器

由于浏览器安全限制（CORS），需通过本地服务器运行此应用。

**方法一：Python（推荐）**

```bash
# Python 3
python3 -m http.server 8000
```

**方法二：VS Code Live Server**

1. 安装 VS Code 扩展：Live Server
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

**方法三：Node.js**

```bash
npm install -g http-server
http-server -p 8000
```

### 2. 打开浏览器

```
http://localhost:8000
```

## 使用说明

### 首页（Dashboard）

- 查看各科目学习统计（完成数、正确率、错题数）
- 点击"开始练习"进入练习模式
- 点击"复习错题"进入复习模式（无错题时按钮禁用）
- 点击"背题模式"进入背题模式
- 使用"导出数据"备份学习进度
- 使用"导入数据"恢复之前的学习数据
- 使用"清除数据"重置所有记录（需确认）

### 练习模式

**鼠标操作：**
- 点击选项选择答案
- 点击"提交"按钮提交答案
- 点击"下一题"/"上一题"导航
- 点击右上角按钮显示/隐藏题目导航面板
- 在导航面板中点击题号直接跳转

**键盘快捷键：**
- `A/B/C/D` - 选择对应选项
- `Enter` - 提交答案（选择题）/ 显示答案（简答题）
- `Space` - 显示答案（简答题）
- `N` - 下一题
- `P` - 上一题
- `V` - 显示/隐藏导航面板
- `Esc` - 退出练习模式

### 复习模式

- 只显示之前答错的题目（不含简答题）
- 答对后自动从错题列表移除
- 使用方式与练习模式相同
- 导航面板只显示错题

### 背题模式

- 直接显示每道题的正确答案（选择题高亮绿色，简答题直接展示）
- 显示题目解析（如有）
- 错题在题目标题前显示 ❌ 标记，导航面板中以红色背景高亮
- **不影响任何统计数据**（不计入完成数、正确率或错题列表）

**键盘快捷键：**
- `N` - 下一题
- `P` - 上一题
- `Esc` - 退出背题模式
- `A/B/C/D` / `Enter` - 无效（背题模式不支持答题操作）

## 题目文件格式

应用读取以下 JSON 文件（放在与 `index.html` 相同目录下）：

- `马克思主义基本原理.json`
- `中国近代史纲要.json`
- `习近平新时代.json`
- `中国近代史纲要_2.json`

### JSON 结构

```json
[
  {
    "questionId": "unique-id",
    "title": "题目内容",
    "type": 0,
    "answerOptions": [
      { "number": 0, "content": "选项A内容" },
      { "number": 1, "content": "选项B内容" }
    ],
    "answer": [0],
    "analysis": "题目解析（可选）",
    "score": 2
  }
]
```

**题目类型：**
- `type: 0` - 单选题
- `type: 1` - 多选题
- `type: 3` - 简答题（`answerOptions` 为空数组，`answer` 为答案文本）

## 数据存储

- 使用浏览器 `localStorage` 存储所有数据
- 存储内容：错题记录、学习统计、答题状态（含选择内容）
- 数据仅保存在本地浏览器，不上传至任何服务器
- 清除浏览器数据会丢失记录（建议定期导出备份）

### 数据结构

| 键名 | 内容 |
|------|------|
| `wrongAnswers` | 各科目错题 questionId 列表 |
| `stats` | 各科目统计（total/completed/correct/wrong） |
| `answeredQuestions` | 各科目答题状态及选择内容 |

## 故障排除

### 题目无法加载

**解决方案：**
1. 确认 JSON 文件在正确位置
2. 确认通过本地服务器运行（不是直接打开 HTML 文件）
3. 检查浏览器控制台的错误信息

### 数据丢失

**解决方案：**
1. 检查是否清除了浏览器缓存/数据
2. 使用"导入数据"功能恢复之前的备份
3. 建议定期使用"导出数据"功能备份

### 键盘快捷键不工作

**解决方案：**
1. 确认当前在练习/复习/背题模式页面
2. 点击页面确保焦点在页面上
3. 检查是否有其他应用占用了相同的快捷键
