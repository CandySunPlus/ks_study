## 修改需求

### 需求:已答选择题展示结果
在练习模式下，当用户进入一道已回答的选择题时，系统必须：
- 选中之前选择的选项并设为 disabled
- 正确选项显示绿色高亮
- 用户选错的选项显示红色高亮
- 显示该题目的 analysis 解析内容
- 隐藏提交按钮
- 禁止修改答案

#### 场景:进入已答对的题目
- **当** 用户导航到一道之前答对的选择题
- **那么** 之前选中的选项以 disabled + 绿色高亮展示，显示解析，提交按钮隐藏

#### 场景:进入已答错的题目
- **当** 用户导航到一道之前答错的选择题
- **那么** 之前选中的选项以 disabled + 红色高亮展示，正确答案以绿色高亮展示，显示解析，提交按钮隐藏

#### 场景:进入未答题目
- **当** 用户导航到一道未回答的题目
- **那么** 按正常流程展示选项和提交按钮，允许作答
