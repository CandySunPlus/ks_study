## 新增需求

### 需求:练习会话选项控制
用户每次进入练习模式时，必须通过弹窗同时选择答题顺序和数据模式，禁止跳过。

#### 场景:选择继续上次进度
- **当** 用户在弹窗中选择"继续上次进度"并点击开始
- **那么** 系统恢复该科目的历史作答记录和统计，按所选顺序进入练习

#### 场景:选择全新开始
- **当** 用户在弹窗中选择"全新开始"并点击开始
- **那么** 系统清除当前科目的 answeredQuestions、wrongAnswers、stats，以空白状态进入练习

#### 场景:取消弹窗
- **当** 用户点击弹窗中的取消按钮
- **那么** 系统返回仪表盘，不进入练习
