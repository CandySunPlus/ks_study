## 为什么

当前练习模式设置显示为小字内联文本加链接样式的"修改"按钮，可见性较低，用户容易忽略。这可能导致用户误用"清除数据"功能来修改练习模式偏好，造成所有学习记录的意外丢失。

## 变更内容

- 将练习模式设置从内联文本样式改为独立的卡片样式
- 使用更显眼的按钮替代链接样式的"修改"文字
- 添加图标和更清晰的视觉层次，提高可发现性
- 保持功能逻辑不变，仅改进 UI 呈现

## 功能 (Capabilities)

### 新增功能
无 - 这是纯 UI 改进，不引入新功能

### 修改功能
无 - 功能行为和需求保持不变，仅改进视觉呈现

## 影响

- **代码文件**:
  - `index.html`: 修改 Dashboard header 中练习模式设置的 HTML 结构，从内联文本改为卡片布局
  - `styles.css`: 添加练习模式设置卡片的样式，使用现有的卡片设计系统（阴影、圆角、渐变）
  - `app.js`: 功能逻辑保持不变，可能需要微调 DOM 选择器

- **用户体验**:
  - 提高练习模式设置的可发现性
  - 降低用户误用"清除数据"的风险
  - 与现有 Dashboard 卡片风格保持一致

- **设计系统**:
  - 复用现有的卡片样式和按钮样式
  - 保持视觉一致性
