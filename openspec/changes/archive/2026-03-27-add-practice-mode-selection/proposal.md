## 为什么

当前刷题工具的练习模式按照题目在 JSON 文件中的固定顺序呈现，用户容易记住题目位置而非真正掌握知识点。添加随机顺序选项可以打破位置记忆，更有效地测试知识掌握程度。

## 变更内容

- 新增练习模式选择功能：用户在进入练习时可选择"顺序练习"或"随机练习"
- 添加模态框 UI，在点击"开始练习"按钮时弹出，让用户选择练习模式
- 支持"记住我的选择"功能，保存用户偏好到 localStorage
- 随机模式下，每次进入练习都重新打乱题目顺序（不保存随机种子）
- 在练习页面显示当前模式状态（顺序/随机）
- 复习模式和背题模式不受影响，保持原有顺序

## 功能 (Capabilities)

### 新增功能
- `practice-mode-selection`: 练习模式选择器，包含模态框 UI、用户偏好存储和模式切换逻辑

### 修改功能
- `practice-mode`: 需要支持随机题目顺序，在加载题目后根据用户选择决定是否打乱顺序

## 影响

- **代码文件**:
  - `app.js`: 修改 `startPractice()` 函数，添加模式选择逻辑和题目打乱功能
  - `index.html`: 添加模态框 HTML 结构
  - `styles.css`: 添加模态框样式

- **存储**:
  - localStorage 新增 `preferredPracticeMode` 键，存储用户偏好（'sequential' | 'random' | null）

- **用户体验**:
  - 进入练习流程增加一步选择环节（如果未保存偏好）
  - 题目导航面板的题号含义变化：随机模式下题号代表当前随机顺序位置，而非原始位置
  - 答题状态持久化、错题追踪等功能不受影响（基于 questionId）
