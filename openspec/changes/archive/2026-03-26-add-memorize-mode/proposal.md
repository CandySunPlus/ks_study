## 为什么

用户需要一个背题模式来更高效地记忆知识点。现有的练习模式需要用户先选择答案再提交才能看到正确答案，这在用户只想快速浏览和记忆所有题目及其答案时效率较低。背题模式将直接显示正确答案和解析，帮助用户专注于记忆而非测试。

## 变更内容

- **新增背题模式入口**：在首页 Dashboard 添加"背题模式"按钮，用户可选择科目进入背题模式
- **新增背题模式视图**：创建独立的背题模式界面，区别于练习模式和复习模式
- **选择题自动显示答案**：进入题目后直接高亮显示正确选项，无需用户交互
- **简答题直接显示答案**：移除"显示答案"按钮，直接进入就展示答案内容
- **显示题目解析**：练习/复习模式提交后显示解析，背题模式下直接显示解析（如果存在）
- **错题标识**：在背题模式下，错题在题目标题处显示特殊标识（如 ❌）
- **导航面板集成**：背题模式下导航面板正常工作，错题格子以红色标识

## 功能 (Capabilities)

### 新增功能
- `memorize-mode`: 背题模式功能，允许用户直接浏览题目及其正确答案和解析，无需交互答题

### 修改功能
- `practice-mode`: 增加题目解析显示功能，提交答案后显示解析内容（如果存在）

## 影响

- **app.js**: 新增背题模式状态管理、渲染逻辑和事件处理
- **index.html**: 可能需要在 Dashboard 添加背题模式按钮
- **styles.css**: 可能需要添加背题模式特定的样式（如错题标识样式）
- **用户数据**: 背题模式不记录答题统计，不影响现有数据存储格式
