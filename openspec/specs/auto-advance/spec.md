# Spec: auto-advance

## 目的

实现选择题答对后的自动跳转功能，提升练习效率。

## 需求

### 需求:答对后自动进入下一题

在练习模式和复习模式下，选择题（单选题和多选题）答对后，系统应当在 300 毫秒后自动进入下一题。

#### 场景:答对选择题后自动跳转

- **当** 用户提交选择题且答案正确
- **那么** 系统在 300ms 后自动加载下一题

#### 场景:答错时保持停留

- **当** 用户提交选择题且答案错误
- **那么** 系统停留在当前题目，显示错误反馈，等待用户手动点击下一题

#### 场景:简答题不适用自动跳转

- **当** 用户提交简答题（type 3）
- **那么** 系统不触发自动跳转，按现有逻辑处理

#### 场景:背诵模式不适用

- **当** 用户处于背诵模式
- **那么** 系统不触发自动跳转，按现有逻辑处理

#### 场景:多选题部分正确时不跳转

- **当** 用户提交多选题但答案不完全正确
- **那么** 系统停留在当前题目，等待用户手动操作
