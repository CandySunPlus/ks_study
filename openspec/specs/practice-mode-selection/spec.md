# practice-mode-selection Specification

## Purpose
练习模式选择功能允许用户在进入练习时选择顺序或随机模式，并支持保存用户偏好以便下次自动使用。

## Requirements

### 需求:模式选择弹窗内容
弹窗必须包含两组互斥选项：
1. 答题顺序：顺序模式 / 随机模式（默认顺序）
2. 答题数据：继续上次进度 / 全新开始（默认继续）

弹窗必须禁止记住选择功能，每次进入练习都必须显示弹窗。

#### 场景:弹窗展示
- **当** 用户点击"开始练习"
- **那么** 弹窗显示两组选项，无"记住选择"checkbox，取消和开始按钮可用

#### 场景:默认选项
- **当** 弹窗打开时
- **那么** 默认选中"顺序模式"和"继续上次进度"

### 需求:模态框操作按钮
模态框必须提供"开始练习"和"取消"两个操作按钮。

#### 场景:点击开始练习
- **当** 用户选择一个模式
- **并且** 用户点击"开始练习"按钮
- **那么** 系统关闭模态框
- **并且** 系统使用选择的模式进入练习页面

#### 场景:点击取消
- **当** 用户点击"取消"按钮
- **那么** 系统关闭模态框
- **并且** 系统保持在 Dashboard 页面

#### 场景:按 ESC 键关闭
- **当** 模态框显示时
- **并且** 用户按下 ESC 键
- **那么** 系统关闭模态框
- **并且** 系统保持在 Dashboard 页面

### 需求:模态框样式
模态框必须符合应用的视觉设计风格。

#### 场景:模态框外观
- **当** 模态框显示时
- **那么** 模态框使用现有的紫蓝渐变主题
- **并且** 模态框居中显示在页面上方
- **并且** 模态框背后显示半透明遮罩层
- **并且** 单选按钮和复选框样式与应用其他表单元素一致

### 需求:复习和背题模式不受影响
复习模式和背题模式禁止显示模式选择模态框。

#### 场景:点击复习错题
- **当** 用户点击某科目的"复习错题"按钮
- **那么** 系统直接进入复习模式
- **并且** 系统不显示练习模式选择模态框

#### 场景:点击背题模式
- **当** 用户点击某科目的"背题模式"按钮
- **那么** 系统直接进入背题模式
- **并且** 系统不显示练习模式选择模态框
