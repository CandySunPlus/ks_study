# practice-mode-selection Specification

## Purpose
练习模式选择功能允许用户在进入练习时选择顺序或随机模式，并支持保存用户偏好以便下次自动使用。

## Requirements

### 需求:模态框显示
当用户点击"开始练习"按钮且未保存练习模式偏好时，系统必须显示模态框让用户选择练习模式。

#### 场景:首次进入练习
- **当** 用户点击某科目的"开始练习"按钮
- **并且** localStorage 中不存在 `preferredPracticeMode` 键或其值为 null
- **那么** 系统显示练习模式选择模态框

#### 场景:已保存偏好时跳过模态框
- **当** 用户点击某科目的"开始练习"按钮
- **并且** localStorage 中存在 `preferredPracticeMode` 键且值为 'sequential' 或 'random'
- **那么** 系统直接使用保存的模式进入练习，不显示模态框

### 需求:模式选择选项
模态框必须提供两个互斥的选项供用户选择：顺序练习和随机练习。

#### 场景:显示两个选项
- **当** 模态框显示时
- **那么** 用户看到"顺序练习"单选按钮及其描述"按题目原始顺序依次练习"
- **并且** 用户看到"随机练习"单选按钮及其描述"每次进入随机打乱题目顺序"
- **并且** 两个选项互斥（只能选择一个）

#### 场景:默认选中顺序练习
- **当** 模态框显示时
- **那么** "顺序练习"选项默认被选中

### 需求:记住选择功能
模态框必须提供"记住我的选择"复选框，允许用户保存其偏好。

#### 场景:勾选记住选择
- **当** 用户勾选"记住我的选择"复选框
- **并且** 用户点击"开始练习"按钮
- **那么** 系统将用户选择的模式（'sequential' 或 'random'）保存到 localStorage 的 `preferredPracticeMode` 键

#### 场景:未勾选记住选择
- **当** 用户未勾选"记住我的选择"复选框
- **并且** 用户点击"开始练习"按钮
- **那么** 系统不保存用户偏好到 localStorage
- **并且** 下次点击"开始练习"时仍会显示模态框

### 需求:模态框操作按钮
模态框必须提供"开始练习"和"取消"两个操作按钮。

#### 场景:点击开始练习
- **当** 用户选择一个模式
- **并且** 用户点击"开始练习"按钮
- **那么** 系统关闭模态框
- **并且** 系统使用选择的模式进入练习页面

#### 场景:点击取消
- **当** 用户点击"取消"按钮
- **那么** 系统关闭模态框
- **并且** 系统保持在 Dashboard 页面

#### 场景:按 ESC 键关闭
- **当** 模态框显示时
- **并且** 用户按下 ESC 键
- **那么** 系统关闭模态框
- **并且** 系统保持在 Dashboard 页面

### 需求:修改保存的偏好
系统必须提供修改已保存偏好的入口。

#### 场景:Dashboard 显示当前默认模式
- **当** 用户查看 Dashboard 页面
- **并且** localStorage 中存在 `preferredPracticeMode` 偏好
- **那么** Dashboard 顶部显示当前默认模式（"顺序练习"或"随机练习"）
- **并且** 显示"修改"按钮

#### 场景:点击修改按钮
- **当** 用户点击"修改"按钮
- **那么** 系统清除 localStorage 中的 `preferredPracticeMode` 键
- **并且** 系统显示练习模式选择模态框

### 需求:模态框样式
模态框必须符合应用的视觉设计风格。

#### 场景:模态框外观
- **当** 模态框显示时
- **那么** 模态框使用现有的紫蓝渐变主题
- **并且** 模态框居中显示在页面上方
- **并且** 模态框背后显示半透明遮罩层
- **并且** 单选按钮和复选框样式与应用其他表单元素一致

### 需求:复习和背题模式不受影响
复习模式和背题模式禁止显示模式选择模态框。

#### 场景:点击复习错题
- **当** 用户点击某科目的"复习错题"按钮
- **那么** 系统直接进入复习模式
- **并且** 系统不显示练习模式选择模态框

#### 场景:点击背题模式
- **当** 用户点击某科目的"背题模式"按钮
- **那么** 系统直接进入背题模式
- **并且** 系统不显示练习模式选择模态框
